# Simple Calculator

A minimal web-based calculator built with HTML, CSS and JavaScript.

## Files
- [index.html](index.html) — UI and markup
- [script.js](script.js) — calculator logic (uses [`buttons`](script.js) and [`string`](script.js))
- [style2.css](style2.css) — main styling
- [styles.css](styles.css) — utility layout classes

## Features
- Basic arithmetic via on-screen buttons
- Clear (C) and equals (=)
- Responsive layout using simple utility classes

## Usage
Open [index.html](index.html) in a browser, or serve the folder (e.g. VS Code Live Server).

Example:
```sh
# if you have http-server installed
npx http-server
# then open http://localhost:8080
